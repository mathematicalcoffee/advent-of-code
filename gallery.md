## Day 20 - conjunctions and flipflops

The default layout alg on the plotting library made the structure much easier to discern.

![](./2023/20/day20-graph-coloured.png)

This is module on the Y axis, iteration on the X axis, and state (on/off = green/red) of each module over iteration as the colour.
You can see the powers-of-2 pattern, but you can **also** see the 'glitch'/period marker at ~3800 iterations where the pattern is irregular and everyone flips to off.
(It is easier in the second zoomed-in picture).

I just thought it looks like a sweet ugly Christmas jumper :P
